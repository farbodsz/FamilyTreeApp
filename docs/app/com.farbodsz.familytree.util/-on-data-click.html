OnDataClick

typealias OnDataClick<T> = (view: View, itemData: T) -> Unit

Type definition for an action to be performed when a data item in a collection of items has been clicked.

This is a function type with two parameters: the View that was clicked/selected, and the corresponding data item from the collection. The function returns Unit.

Parameters

T - the type of data item
