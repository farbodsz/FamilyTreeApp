drawNodeAndChildren

private fun drawNodeAndChildren(canvas: Canvas, node: TreeNode<Person>, depth: Int = 0, parentXLeft: Int = 0, parentCentre: Pair<Float, Float>? = null, childPos: Int = 0): Unit

Draws a representation of a node and its children onto the canvas.

Parameters

canvas - the canvas to draw to (parameter from View.onDraw)

node - the node to be drawn

depth - the depth of the node being drawn (0 by default)

parentXLeft - X coordinate of the left of the space allocated for drawing this node's parent (0 by default, indicating this node has no parent)

parentCentre - pair of float coordinates (X then Y) of the bottom centre of this node's parent's drawn area. It can be null, indicating no parent.

childPos - a number representing this (child) node's position in relation to its siblings starting from 0. (0 is also used by default to indicate this node has no parent, i.e. is the root).
