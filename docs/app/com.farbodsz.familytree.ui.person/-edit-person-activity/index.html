EditPersonActivity

class EditPersonActivity : AppCompatActivity

This activity provides the UI for adding or editing a new person from the database.

When the user adds all the information and confirms, the data for the new person will be written to the database, and the newly created Person will be sent back to the activity from which this was started as a result.

This activity is not responsible for creating a person: for such cases, CreatePersonActivity should be started for result (it will return the new object to the calling activity).
