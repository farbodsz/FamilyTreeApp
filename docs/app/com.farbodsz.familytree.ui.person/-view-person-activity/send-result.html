sendResult

private fun sendResult(): Unit

Sends the correct result back to where this activity was invoked from, and finishes the activity.

An "ok" result will be used if the person has been modified, otherwise a "cancelled" result. This function is not concerned with "successful" results that indicate the deletion of a person. Those are handled in showDeleteDialog.
